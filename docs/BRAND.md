# Subscription Dock Brand Guide

This guide keeps the project visually consistent across GitHub, the product UI, and release notes.

## Brand idea

Subscription Dock is the calm place where recurring expenses become visible and stay under control. The visual language is compact, dark, and data-first.

## Core palette

| Token | Hex | Use |
|---|---:|---|
| Dock Black | `#0B0C0F` | Primary background |
| Panel | `#15171C` | Cards and elevated surfaces |
| Border | `#2A2D34` | Dividers and outlines |
| Dock Amber | `#F59E0B` | Primary accent and calls to action |
| Dock Teal | `#2DD4BF` | Logo accent |
| Dock Rose | `#FB7185` | Logo accent and destructive emphasis |
| Ice White | `#F7F8FA` | Primary text |
| Harbor Gray | `#9DA3AE` | Secondary text |

## Logo usage

- Keep clear space equal to one quarter of the icon height.
- Pair the square mark with a live-text “Subscription Dock” heading in documentation.
- Use the icon alone for favicons and avatars.
- Do not recolor, add drop shadows, stretch, or place it on a busy background.

## Typography

Use Inter when available, then `Segoe UI`, Arial, or a system sans-serif fallback. Product copy should be direct, calm, and practical.
